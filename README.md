# Tsugiki (接ぎ木)

HTML tree-manipulation library for Rust.

[Documentation](https://docs.rs/kuchikiki)

This is a fork of the Kuchiki (朽木) library, which in now unmaintained, based on the Kuchikiki (口利き) branch by the Brave team.

You can use this version by updating the name in your `Cargo.toml` (replacing `kuchiki` or `kuchikiki` with `tsugiki`), then remap code references to the new name, e.g. with:

```rust
use tsugiki as kuchiki;
```

See the [Security Policy](SECURITY.md) for information on reporting vulnerabilities.
